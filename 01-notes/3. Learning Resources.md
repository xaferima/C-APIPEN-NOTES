# Learning Resources and Practice Environments

Suggested official and independent resources for exam preparation:

### 🎓 Learning Platforms and Theory
- **PortSwigger:** [API Testing Learning Path](https://portswigger.net/web-security/learning-paths/api-testing) (Free, excellent for Burp Suite, IDOR, SSRF).
- **Kontra:** [OWASP Top 10 API](https://application.security/free/owasp-top-10-API) (Free, interactive simulations).
- **TryHackMe:** [OWASP API Security Top 10](https://tryhackme.com/r/room/owaspapisecuritytop105w) (Free, practical step-by-step).
- **HackTheBox / PentesterLab:** Specific API exploitation exercises.

### 🐳 Local Vulnerable Environments (Goats)
You can deploy these labs locally with Docker to practice API exploitation:
- [Node API Goat](https://github.com/layro01/node-api-goat) - Vulnerable lab in Node.js.
- [Rest API Goat](https://github.com/optiv/rest-api-goat) - Vulnerable REST API for penetration testing.
- [Vapi](https://github.com/roottusk/vapi) - Vulnerable API with OWASP API Top 10 preconfigured.
- [Vulnapi](https://github.com/tkisason/vulnapi) - Tool and vulnerable API for scanning and testing.
- [SQLI-Labs](https://github.com/Audi-1/sqli-labs) - Excellent for practicing SQL injections applicable to REST endpoints.
