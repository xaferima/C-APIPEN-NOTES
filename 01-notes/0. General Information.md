# Certified API Pentester (C-APIPen)

The **Certified API Pentester (C-APIPen)** exam from *The SecOps Group* is a practical intermediate-level certification designed to evaluate hands-on skills in API security audits (REST, SOAP, and GraphQL).

## 📊 Key Exam Details
- **Official Duration:** 04:15 (4 hours and 15 minutes).
- **Number of Questions:** 8 questions.
- **Format:** Exploitation of vulnerabilities and capture of flags in a real environment.
- **Price:** Original £250. **£63** using discount code `75-OFF`.
- **Passing Criteria:**
  - **Passed with Merit:** 75.00% to 100.00%.
  - **Passed:** 60.00% to 74.99%.
  - **Failed:** 00.00% to 59.99%.
- **Validity and Retake:** Does not expire. Includes 1 free retry attempt.
